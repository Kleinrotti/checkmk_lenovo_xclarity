# CheckMK Agent for Lenovo XClarity (SNMP)

## Requirements

- CheckMK >=2.3.0p1

## How to use

- Install the mkp package in Checkmk. [HowTo](https://docs.checkmk.com/latest/en/mkps.html)
- Do a service discovery

## What is monitored

- Fans
- PSUs
- Temperatures
- Voltages
- Disks
- Power consumption
- General system health
